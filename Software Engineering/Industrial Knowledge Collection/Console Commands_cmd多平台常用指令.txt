****************************
Linux/Unix
****************************
mkdir <path> 在当前位置创建路径/文件path
cd <path> 移动当前工作目录至path
pwd 显示当前路径名
ls 显示当前路径的文件内容
cat <path>
rm <path> 删除路径
****************************
Windows
****************************
dir 显示目录下的文件及子目录
cd <path> 移动当前工作目录至path
findstr <string> <filepath> 在目录下查找指定字符内容，有诸多选项参数，document内容存疑，使用findstr /?指令获取现场帮助
