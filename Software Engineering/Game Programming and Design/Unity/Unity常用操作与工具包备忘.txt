通用
Ctrl + P ：运行游戏项目play
Alt+左键拖拽：旋转视图orbit
左键拖拽：平移视图pan
w：切换至移动模式move tool
q：切换至手型模式hand tool
e：切换至旋转模式rotate tool
r：切换至缩放模式scale tool
f：聚焦所选物体
菜单-Game Object-Align with View：将所选GO对齐Scene的视窗，一般用于调整camera
*********************************************************************************
Curve Edit
Ctrl + Zoom（如滚轮）：水平缩放，调整x轴的视野
Shift + Zoom：垂直缩放，调整y轴的视野
